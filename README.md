# PinnedListView
    本项目引用了开源项目https://github.com/beworker/pinned-section-listview

---
仅仅是将原先的项目进行个人改造封装，更适合个人扩展而已。

=====
##quick start
- 继承PinnedBaseAdapter类，实现getSectionLayout和getItemLayout方法，分别返回section的布局和item的布局。

- 在getview方法中进行个性化的内容返回。
